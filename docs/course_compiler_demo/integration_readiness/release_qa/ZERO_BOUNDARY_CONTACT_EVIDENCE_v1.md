# Zero Boundary Contact Evidence v1

Every reproduction result records false boundary assertions for:

- adaptive-platform access, modification, Git activity, staging, commit preparation, commit creation, push, and staging-directory writes
- database contact
- network endpoint contact
- runtime-route calls
- learner-loop work
- grading, persistence, and retrieval work
- canonical promotion
- student-visible output
- source package modification

The evidence also records compiler repository root, allowed output root, commands executed, files read, files written, consulted environment-variable names, and imported compiler modules. Secret values are not recorded.

Boundary evidence is generated by the compiler-side harness only. It does not inspect adaptive-platform.
